# Helios Documentation Theme

Welcome to Helios, a modern documentation theme for Grav CMS built on **Tailwind CSS 4** and **Alpine.js**.

## Features

Helios provides everything you need to create beautiful, professional documentation:

- **Three-column layout** - Sidebar navigation, content area, and on-page table of contents
- **Dark/Light mode** - System preference detection with manual toggle
- **Full-text search** - Powered by SimpleSearch with keyboard shortcuts (Cmd+K)
- **API documentation** - Method badges, parameter tables, request/response examples
- **Folder-based versioning** - Organize docs by version with easy switching
- **Code highlighting** - Syntax highlighting with copy button support
- **Responsive design** - Works beautifully on desktop, tablet, and mobile

## Quick Start

Get started with Helios in just a few steps:

```bash
# Install the theme
bin/gpm install helios

# Set as active theme
bin/grav theme helios
```

Or manually download and extract to your `user/themes/helios` folder.

## Navigation

Use the sidebar on the left to explore the documentation:

- **Getting Started** - Installation, configuration, and quick start guide
- **Guides** - In-depth guides for theming, navigation, and versioning
- **API Reference** - Example API documentation pages
